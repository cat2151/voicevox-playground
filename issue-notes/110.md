# issue 画面の右上に「設定」ボタンを配置し、折りたたみで設定が開くようにし、port番号「50021」と、周波数推定対象上位%「2」、を設定可能にする #110
[issues #110](https://github.com/cat2151/voicevox-playground/issues/110)

# 設定内容
- VOICEVOXのport番号：「50021」がデフォルト
- VOICEVOX nemo のport番号：「50121」がデフォルト
- 周波数推定対象上位%：「2」がデフォルト
- 折りたたみ内部に、「初期値に戻す」ボタンもつける

# 保存と復帰
- 変更時はlocal storageに保存する
- 起動時は、local storageから復帰する。local storageが空ならデフォルト値を設定する（「初期値に戻す」ボタンと同様の挙動）
