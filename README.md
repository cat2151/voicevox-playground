## Features

- Send requests to the VOICEVOX local HTTP server (port 50021)
- Read out any text using Zundamon's voice
- Audio playback using Tone.js v15
- Deployed to GitHub Pages

## Prerequisites

The VOICEVOX local server must be running.

1.  Download and install [VOICEVOX](https://voicevox.hiroshiba.jp/)
2.  Start the VOICEVOX engine (an HTTP server will start on port 50021). If accessing from the GitHub Pages version (`https://cat2151.github.io/voicevox-playground/`), use the following command with CORS enabled:

    ```bash
    <your VOICEVOX directory>/vv-engine/run --cors_policy_mode all --allow_origin https://cat2151.github.io
    ```

    If you also want to use it from the local development server (provided by `npm run dev` at `http://localhost:5173`), append `http://localhost:5173` to the above command:

    ```bash
    <your VOICEVOX directory>/vv-engine/run --cors_policy_mode all --allow_origin https://cat2151.github.io http://localhost:5173
    ```

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build
npm run build

# Preview
npm run preview
```

## Deployment

Automatically deployed to GitHub Pages using GitHub Actions.

1.  In your repository's Settings > Pages, set Source to "GitHub Actions".
2.  Pushing to the main branch will trigger an automatic deployment.

## Usage

1.  Start VOICEVOX.
2.  Open the application in your browser.
3.  Enter the text you want to be read aloud into the text area.
4.  Click the "Play" button.

## Tech Stack

- TypeScript
- Vite
- Tone.js v15
- VOICEVOX API
