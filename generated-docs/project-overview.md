# Project Overview

## プロジェクト概要
- VOICEVOXローカルサーバーと連携し、テキストを音声に変換してWebブラウザで再生するアプリケーションです。
- ユーザーは任意のテキストを多様なキャラクターボイスで読み上げさせ、イントネーションを視覚的に編集できます。
- 本プロジェクトは、WebブラウザからVOICEVOXを手軽に操作できるクライアントアプリの実証を目指しています。

## 技術スタック
- フロントエンド: **TypeScript** (JavaScriptに静的型付けを追加し、大規模開発を支援), **Vite** (高速な開発サーバーとモダンなビルドプロセスを提供する次世代フロントエンドツール)
- 音楽・オーディオ: **Tone.js v15** (Web Audio APIを抽象化し、ブラウザ上での高度な音声合成・処理・再生を可能にするライブラリ), **VOICEVOX API** (VOICEVOXエンジンと連携し、音声合成機能を提供するREST API)
- 開発ツール: **@biomejs/biome** (コードのフォーマット、リンティング、修正を行う高速な開発ツール), **jsdom** (Node.js環境でブラウザのようなDOM環境をシミュレートし、DOM操作を含むテストを可能にする)
- テスト: **vitest** (Viteをベースにした高速で機能豊富なユニットテストフレームワーク)
- ビルドツール: **Vite** (開発サーバーとしての機能に加え、プロダクション向けの最適化されたビルドを生成)
- 言語機能: **TypeScript** (JavaScriptのスーパーセットとして、型安全性と開発生産性を提供)
- 開発標準: **@biomejs/biome** (コードの品質と一貫性を維持するための規約適用)

## 関数詳細説明
- **getAudioQuery** (src/audio.ts):
    - 役割: VOICEVOX APIからテキストに対するAudioQuery（音声合成に必要な情報）を取得します。
    - 引数: `text: string`, `styleId: number`
    - 戻り値: `Promise<any>` (AudioQueryオブジェクト)
- **synthesize** (src/audio.ts):
    - 役割: 取得したAudioQueryを元にVOICEVOX APIから音声データを合成し、WAV形式で取得します。
    - 引数: `audioQuery: any`, `styleId: number`
    - 戻り値: `Promise<Blob>` (音声データのBlob)
- **combineAudioBuffers** (src/audio.ts):
    - 役割: 複数のAudioBufferを一つのAudioBufferに結合します。
    - 引数: `buffers: AudioBuffer[]`
    - 戻り値: `AudioBuffer` (結合されたAudioBuffer)
- **encodeAudioBufferToWav** (src/audio.ts):
    - 役割: AudioBufferの内容をWAV形式のバイナリデータ（Uint8Array）に変換します。
    - 引数: `audioBuffer: AudioBuffer`
    - 戻り値: `Uint8Array` (WAV形式のバイナリデータ)
- **writeString** (src/audio.ts):
    - 役割: 指定された文字列をDataViewの指定オフセットに書き込みます。WAVヘッダーの構築などに利用されます。
    - 引数: `view: DataView`, `offset: number`, `string: string`
    - 戻り値: なし
- **clamp** (src/audio.ts):
    - 役割: 数値を指定された最小値と最大値の範囲に制限（クランプ）します。
    - 引数: `value: number`, `min: number`, `max: number`
    - 戻り値: `number` (クランプされた数値)
